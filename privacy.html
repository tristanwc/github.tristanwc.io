Privacy Policy

Last updated: March 4, 2026

Overview

DrawPose is a personal drawing practice tool. It reads public Pinterest boards to display reference images for timed drawing sessions. No user data is collected, stored, or shared.

Pinterest API Usage

Croquis uses the Pinterest API solely to fetch publicly available pin images from boards you provide. No Pinterest account credentials are accessed. No data from Pinterest is retained beyond your browser session.

Data Collection

This application does not collect, store, or transmit any personal information. There are no accounts, no analytics, and no cookies.
